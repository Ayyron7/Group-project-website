# Karaoke Library System
A web-based karaoke management system that allows users to browse songs, request their favorites, and manage the singing queue.

## Features

- **Song Library**: Browse through an extensive collection of karaoke songs
- **Song Requests**: Users can request songs they want to perform
- **Queue Management**: Add yourself to the performance queue
- **Real-time Updates**: Track your position in the queue

## Technologies Used
- PHP
- MySQL/Database
- HTML/CSS
- C++
